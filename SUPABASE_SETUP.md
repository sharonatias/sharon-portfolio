# הוראות הוספת עמודת Order לטבלת hero_videos בSupabase

## שלב 1: כניסה לSupabase Dashboard
1. פתח את הדפדפן וגש ל: https://supabase.com
2. התחבר עם האימייל והסיסמה שלך
3. בחר את הפרויקט **sharon-portfolio**

## שלב 2: פתיחת SQL Editor
1. בתפריט הימני של הדאשבורד, חפש את **SQL Editor**
2. לחץ על **SQL Editor** (זה יהיה כתוב בעברית: "עורך SQL")

## שלב 3: יצירת Query חדש
1. בחלק העליון, לחץ על **New Query** (או "Query חדש")
2. תוך כדי זה, פתח חלון עריכה ריק

## שלב 4: הדבקת ה-SQL
העתק והדבק את הטקסט הזה בעריכה:

```sql
ALTER TABLE hero_videos 
ADD COLUMN IF NOT EXISTS "order" bigint DEFAULT 0;

UPDATE hero_videos
SET "order" = ROW_NUMBER() OVER (ORDER BY created_at ASC) - 1
WHERE "order" IS NULL OR "order" = 0;
```

## שלב 5: הרצת ה-Query
יש שתי דרכים:

**דרך 1 - לחצן Run:**
- חפש כפתור שחור או כחול בפינה הימנית-עליונה של העורך
- הכפתור כתוב עליו "Run" או יש עליו סימן משחק ▶
- לחץ על הכפתור הזה

**דרך 2 - קיצור מקלדת:**
- אם אתה ב-Mac: לחץ **Cmd + Enter**
- אם אתה ב-Windows/Linux: לחץ **Ctrl + Enter**

## שלב 6: בדיקה שההוספה הצליחה
- בחלק התחתון של המסך, תראה הודעה בירוק שאומרת: **"Success"** או משהו דומה
- זה אומר שה-Query הצליח!

## שלב 7: אישור שהעמודה נוספה
1. בתפריט הימני, לחץ על **Table Editor** (או "עורך טבלאות")
2. בחר את הטבלה **hero_videos**
3. בדוק שיש עמודה חדשה שנקראת **order**
4. אם אתה רואה אותה - הכל הצליח! ✓
